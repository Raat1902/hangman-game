# Hangman Game (Java)

A simple console-based **Hangman** game written in **Java**.

## Features
- Random word selection from a word bank
- Tracks correct and wrong guesses
- Displays remaining lives
- Prevents guessing the same letter twice
- Beginner-friendly console UI
